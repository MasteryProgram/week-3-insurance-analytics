# AlphaCare Insurance Risk Analytics

## Project Overview
End-to-End Insurance Risk Analytics & Predictive Modeling for AlphaCare Insurance Solutions (ACIS), South Africa.

**Objective**: Identify low-risk customer segments and build risk-based pricing models.

## Project Structure
- `data/` → Raw and processed datasets
- `notebooks/` → EDA and analysis
- `src/` → Reusable Python modules
- `reports/` → Interim & Final reports

## How to Reproduce
1. Clone the repo
2. `pip install -r requirements.txt`
3. `dvc pull`
4. Run `01_eda.ipynb`
